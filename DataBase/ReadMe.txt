В файле Data хранится информация о людях в формате:
Имя : ДД.ММ.ГГГГ

Сама программа выполняет такие функции, как:
1. Change, которая запрашивает имя человека и ждет дальнейшего выбора между Data и Name.
Если такого имени нет, то программа сообщит об этом.
Если Вы хотите сменить имя, то вводите Name. Потом программа ожидает Вашего нового имени, на которрое Вы хотите 
сменить имя, запрашиваемое ранее.
Если Вы хотите сменить дату рождения, то вводите Data. Программа будет ожидать нового дня рождения для 
выбранного Вами имени.
2. Delete, которая удаляет человека из базы, если он там находится на данный момент.
На вход принимает лишь имя.
3. Age, которая говорит о том, сколько сейчас лет человеку.
Принимает на вход имя, которое сразу же проверяется на наличие в базе данных.
4. B-day, которая сообщает о том, у кого скоро День Рождения.
5. Inf, которая позволяет найти человека по его дню, месяцу и году рождения.
Функция спрашивает о том, по какому параметру Вы хотите найти человека:
-day, запрашивает день и месяц рождения. Проверяет на правильность указания этих параметров.
-month, запрашивает месяц рождения. Проверяет на правильность указания параментра.
-year, запрашивает год рождения. Проверяет на правильность, сравнивая с текущим годом с одной стороны 
и с 0 года нашей эры с другой.
6. Add, которя добавляет человека в базу данных. На вход принимает имя и день рождения. Если дата введена
неправильно, то программа сообщит об этом.
7. Quit, которая позволяет выйти из программы, при этом записав в исходный файл все изменения, которые производились
в ходе выполнения программы. 

PS.Программа будет выполняться до тех пор, пока не будет введено "Quit" в поле "Enter the command: ".

В программе производятся проверки на правильность введенных дат; на наличие человека, с которым Вы хотите взаимодествовать
в текущей функции, в базе данных; на наличие введенной команды.
Так же, если в файле есть имя с ":", то оно в последствие уйдет из файла,потому что оно некорректно,
поэтому данное имя не будет проверяться во всех командах программы.
Так же, если в файле день рождения записан некорректно, то есть не может существовать такого дня, месяца, года, то в начале
программы данный человек будет удален.
